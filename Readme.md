# Air Quality Index
## 
A standardised metric that converts multiple pollutant measurements into a single score to indicate how clean or polluted the air is and the associated health implications.

## Features
- Data Input and Management
- Data Display
- Search Functionality
- Analytical Features
- Menu-Driven Interface
- Robust Data Structure
- Error Handling and Validation

## Tech

The AQI Application is developed using the following technologies & tools:


- Python 3
- VSCODE
- Version Control (GIT)
- [node.js] - evented I/O for the backend

## Installation

AQI requires [Python 3+](https://www.python.org/) to run.

After installation.

```sh
cd AirQualityIndex/ 
python3 main.py/
✨Magic ✨
```
